# CAEP_analysis
基于streamlit搭建的快速可视化脑电信号数据
# Usage
## step 1
安装stream_lit `pip install streamlit`
## step 2
clone this project `git clone https://github.com/Me-gumin/CAEP_analysis.git`
## step 3
运行app.py
<br>1.first `cd CAEP_analysis/util`</br>
<br>2.then `streamlit run app.py`</br>
